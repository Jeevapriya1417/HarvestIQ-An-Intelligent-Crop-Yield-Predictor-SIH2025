## 🔧 Development & Deployment

### **Prerequisites**
- Node.js 18+ 
- npm or yarn package manager

### **Installation**
```bash
# Navigate to project directory
cd HarvestIQ

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### **Environment Setup**
The application includes mock government data services that simulate real API responses. For production deployment, replace with actual government API endpoints.
